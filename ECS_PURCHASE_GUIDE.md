# 阿里云 ECS 购买页面选项指南

> 本文档详细说明阿里云 ECS 购买页面所有选项的推荐配置，适用于 Mini-Shopify Antom 项目部署。

---

## 一、购买入口

1. 登录阿里云控制台：https://ecs.console.aliyun.com
2. 点击左侧菜单「实例与镜像」→「实例」
3. 点击「创建实例」按钮

---

## 二、基础配置

### 2.1 付费模式

| 选项 | 说明 | 推荐 |
|------|------|------|
| **包年包月** | 预付费，价格优惠，适合长期使用 | ⭐ 生产环境推荐 |
| **按量付费** | 后付费，按小时计费，随时释放 | ⭐ 测试环境推荐 |
| 抢占式实例 | 价格最低，但可能被回收 | 不推荐 |

**选择建议：**
- 测试/短期使用：选择「按量付费」
- 生产/长期使用：选择「包年包月」（1年起，折扣多）

---

### 2.2 地域

| 地域 | 说明 | 推荐 |
|------|------|------|
| **华东1（杭州）** | 国内核心节点，延迟低 | ⭐ 推荐 |
| **华东2（上海）** | 国内核心节点 | ⭐ 推荐 |
| 华北2（北京） | 北方用户优先 | 可选 |
| 华南1（深圳） | 南方用户优先 | 可选 |
| 中国香港 | 无需备案，国际线路 | 外贸业务可选 |
| 海外地域 | 新加坡、日本等 | 海外业务可选 |

**选择建议：**
- 国内用户：选择「华东1（杭州）」或「华东2（上海）」
- 无备案需求：选择「中国香港」

**注意：** 地域选择后不可更改！

---

### 2.3 可用区

| 选项 | 说明 | 推荐 |
|------|------|------|
| **随机分配** | 系统自动选择可用区 | ⭐ 推荐 |
| 指定可用区 | 手动选择具体可用区 | 多实例同 VPC 时使用 |

**选择建议：** 单实例选择「随机分配」即可

### 2.4 实例规格

#### 规格族选择

| 规格族 | 适用场景 | 推荐 |
|--------|---------|------|
| **通用型 g系列** | CPU/内存均衡，企业级应用 | ⭐ 推荐 |
| 计算型 c系列 | CPU 密集型应用 | 不推荐 |
| 内存型 r系列 | 内存密集型应用 | 不推荐 |
| 突发性能型 t系列 | 低成本，有 CPU 积分限制 | 仅测试可用 |
| 共享型 n系列 | 入门级，性能受限 | 不推荐 |

#### 具体规格推荐

| 场景 | 推荐规格 | vCPU | 内存 | 价格参考（月） |
|------|---------|------|------|---------------|
| **生产环境** | ecs.g6.large | 2核 | 4GB | 约 ¥150-200 |
| **测试环境** | ecs.g6.xlarge | 2核 | 4GB | 约 ¥150-200 |
| **最低配置** | ecs.t6-c1m2.large | 2核 | 2GB | 约 ¥60-80 |
| **高配环境** | ecs.g6.2xlarge | 4核 | 8GB | 约 ¥300-400 |

**选择建议：**
- 测试环境：`ecs.g6.large`（2核4G）
- 生产环境：`ecs.g6.2xlarge`（4核8G）或更高

### 2.5 镜像

#### 镜像类型选择

| 类型 | 说明 | 推荐 |
|------|------|------|
| **公共镜像** | 官方纯净系统 | ⭐ 推荐 |
| 自定义镜像 | 基于已有实例创建 | 有现成镜像时使用 |
| 共享镜像 | 其他账号共享 | 不推荐 |
| 镜像市场 | 预装软件的镜像 | 可选，省去安装步骤 |

#### 操作系统推荐

| 操作系统 | 版本 | 推荐理由 |
|---------|------|---------|
| **Ubuntu** | **22.04 64位** | ⭐ 首选，Node.js 兼容性最好 |
| Ubuntu | 20.04 64位 | 稳定，LTS 版本 |
| Alibaba Cloud Linux | 3.2104 64位 | 阿里云优化，免费 |
| CentOS | 7.9 64位 | 企业常用，稳定性好 |
| CentOS | 8.x 64位 | 新版本（注意生命周期） |

**选择建议：**
- **推荐 Ubuntu 22.04 64位**
- 路径：公共镜像 → Ubuntu → 22.04 64位

### 2.7 存储

#### 系统盘

| 选项 | 说明 | 推荐 |
|------|------|------|
| **ESSD 云盘** | 高性能 SSD | ⭐ 推荐 |
| 高效云盘 | 性能一般，价格低 | 不推荐 |
| 普通 云盘 | 性能最差 | 不推荐 |

**系统盘容量：**
- 最小 20GB
- 推荐配置：**40GB**（足够系统 + 应用）
- 最大 500GB

**系统盘性能：**
| 性能级别 | IOPS | 适用场景 | 推荐 |
|---------|------|---------|------|
| PL0 | 10,000 | 测试环境 | 可选 |
| **PL1** | 50,000 | 生产环境 | ⭐ 推荐 |
| PL2 | 100,000 | 高性能需求 | 不需要 |
| PL3 | 1,000,000 | 极高性能需求 | 不需要 |

#### 数据盘（可选）

| 是否添加 | 说明 | 推荐 |
|---------|------|------|
| 不添加 | 数据存系统盘 | 测试环境可用 |
| **添加** | 数据与系统分离 | ⭐ 生产环境推荐 |

**数据盘配置（如需添加）：**
- 类型：ESSD 云盘
- 容量：20-100GB
- 性能级别：PL1

**选择建议：**
- 测试环境：系统盘 40GB，无数据盘
- 生产环境：系统盘 40GB + 数据盘 50GB

## 八、购买后操作

### 8.1 查看实例信息

购买完成后：
1. 进入 ECS 控制台 → 实例列表
2. 找到刚创建的实例
3. 记录以下信息：
   - **公网 IP**：用于 SSH 连接和访问
   - **实例 ID**：用于工单和技术支持
   - **内网 IP**：用于内网通信

### 8.2 连接实例

**Windows 用户：**
```bash
# 使用 PowerShell 或 Git Bash
ssh root@你的公网IP
```

**Mac/Linux 用户：**
```bash
ssh root@你的公网IP
```

**首次连接提示：**
```
The authenticity of host 'xxx.xxx.xxx.xxx' can't be established.
Are you sure you want to continue connecting (yes/no)?
```
输入 `yes` 并回车，然后输入密码。

### 8.3 验证环境

```bash
# 检查系统版本
cat /etc/os-release

# 检查内存
free -h

# 检查磁盘
df -h

# 检查网络
ping aliyun.com
```

## 九、常见问题

### Q1: 按量付费和包年包月如何选择？

| 场景 | 推荐 |
|------|------|
| 测试几天就释放 | 按量付费 |
| 长期使用（3个月以上） | 包年包月 |
| 不确定使用时长 | 先按量付费，后续转包年包月 |

### Q2: 带宽选多少合适？

| 用途 | 带宽 | 说明 |
|------|------|------|
| 纯 API 服务 | 1-3 Mbps | 足够 |
| 有前端页面 | 3-5 Mbps | 推荐 |
| 图片/视频较多 | 10 Mbps+ | 按需选择 |

### Q3: 数据盘必须吗？

- **测试环境**：不需要，数据存系统盘即可
- **生产环境**：推荐添加，便于数据备份和迁移

### Q4: 选错了能改吗？

| 配置项 | 能否修改 | 修改方式 |
|--------|---------|---------|
| 地域 | ❌ 不能 | 需重新购买 |
| 实例规格 | ✅ 可以 | 控制台「变更实例规格」 |
| 公网带宽 | ✅ 可以 | 控制台「升配」或「降配」 |
| 系统盘 | ✅ 可以 | 控制台「扩容」 |
| 镜像 | ✅ 可以 | 重装系统（数据会丢失） |

### Q5: 如何省钱？

1. 选择「包年包月」享受折扣
2. 关注阿里云活动（双11、618 等）
3. 使用「抢占式实例」（测试环境）
4. 合理选择带宽，避免浪费
